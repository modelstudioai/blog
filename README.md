## What You'll Find Here

We publish in four categories:

- **Guides** — Step-by-step walkthroughs for tools and workflows. Get productive quickly without reading the full manual.
- **Tutorials** — Deep dives into building real-world AI applications. Code-first, opinionated, and production-aware.
- **Insights** — Analysis of emerging protocols, frameworks, and ecosystem shifts. We connect the dots so you don't have to.
- **Case Studies** — How teams solve specific problems with LLMs. Real constraints, real trade-offs, real outcomes.

## Contributing

We welcome contributions from the community. Whether it's a typo fix, a translation, or an entirely new tutorial — every PR matters.

- **Found an error?** Open an issue or submit a PR directly.
- **Want to propose a new article?** Open an issue with the `article-proposal` label and we'll discuss the angle.
- **Translations** are highly encouraged. If you translate an article, keep the original author's credit and add your own as translator.

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.
