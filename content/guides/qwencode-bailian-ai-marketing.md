## 先说工具：Qwen Code 和百炼 CLI 是什么？

简单理解：

- **Qwen Code** 是一个 AI 编程助手，但它不只能写代码——它是一个通用的 AI Agent，能理解自然语言指令，帮你调用各种工具完成任务
- **百炼 CLI**（`bl` 命令）是阿里云百炼平台的命令行工具，集成了文生图、图生图、文生视频、图生视频等 AI 能力

把它们组合起来，你就拥有了一个**能听懂人话的营销素材生产线**。

### 你需要准备什么？

- 安装 Qwen Code CLI：[Qwen Code 概览 \| Qwen Code Docs](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/)
- 安装百炼 CLI：[阿里云百炼 CLI（Bailian CLI）\| 一行指令调用阿里云百炼全模态 AI 能力](https://bailian.console.aliyun.com/cli?source_channel=cli_github&)
- 一个阿里云百炼 API Key（[百炼控制台](https://bailian.console.aliyun.com/cn-beijing/?source_channel=key_github&tab=app#/api-key) 申请）

准备好了？我们开始。

现在只需要一句话描述你要什么图，AI 直接生成。白底主图、场景图、促销氛围图、科技感主图——全都能出。

### 支持的图片类型

| 类型     | 尺寸                 | 适用场景           |
| ------ | ------------------ | -------------- |
| 电商主图   | 1024×1024（方图 1:1）  | 淘宝/京东商品首图、促销主图 |
| 电商详情页图 | 1024×1366（竖版 3:4）  | 详情页单屏展示        |
| 手机启动页  | 1080×1920（竖版 9:16） | App 开屏广告       |
| 横幅广告   | 1344×756（横版 16:9）  | PC 端 Banner    |

所有尺寸都支持**带营销文字**的商品图——AI 先生成纯视觉产品底图，再用 HTML 精确叠加中文卖点、slogan、价格等文字，最终导出为一张完整的 PNG。文字清晰不变形，改文案只需 2 秒重新渲染，不用重新生图。

## 场景二：飞书里拍张照片，自动出商品图和视频 📸

电商商品图拍摄成本高——租摄影棚、请摄影师、后期修图，一套下来费用几千起步，周期一到两周。小团队根本负担不起。现在各平台还在推短视频，商品视频又是一笔开销。

现在最爽的做法是：**手机拍一张照片，发到飞书群里 @机器人，AI 自动识别你拍的是什么商品，然后一条龙生成白底商品图、带文字的营销主图、5 秒展示视频。**

整个过程不用开电脑。

**第一步：拍照发群**

手机随手拍一张产品的照片（背景随意，办公桌上、沙发上都行），发到飞书群里，@Qwen Code 说：

```
使用 open1v-product-visual 技能帮我生成2张商品图
```

机器人会自动做 3 件事：

1. **识别商品主体** — 自动判断你拍的是什么
2. **生成标准商品图** — 根据商品主体判断生成，存入项目文件夹
3. **返回结果** — 可直接返回云空间图片链接

**第二步：选图，加上营销文字**

选一张满意的白底图，接着说：

```
现在我需要加上营销文字，小红书3:4尺寸，布局你可以自选，我要白底的。
```

机器人根据识别到的商品信息，自动提炼营销文字并叠加到图上：主标题、副标题、卖点标签、规格数据——排版好的成品图直接可以上架。

改文字和布局？直接说：

```
把图片整体向右侧移动一点，产品主体居中，文字和排版不变
```

改文字不需要重新生图，**2 秒钟**重新渲染就出来了。

**第三步：直接返回成品链接**

```
现在我需要你生成图片的链接，上传到云空间并返回给我链接，只需要终稿版本
```

## 场景三：AI 帮你写推广文案 ✍️

写文案是最耗时间的环节之一。公众号要深度长文、小红书要种草短文、详情页要卖点文案——**同一个产品，三个渠道，三种写法**。每次都从零开始写，写到最后词穷了，翻来覆去就那几句话。

现在只需要告诉 AI 产品信息和投放渠道，公众号写故事、小红书写种草、详情页写卖点——一次输入，多渠道输出。

**例子：为一款便携充电宝生成 3 个渠道的推广文案**

在 Qwen Code 中输入：

```
帮我写一篇充电宝的推广文案。

产品名：PowerMax 迷你充电宝

卖点：1. 仅重 120g，比一颗鸡蛋还轻 2. 10000mAh 大容量 3. 支持 65W 快充，30 分钟充满 iPhone

目标人群：经常出差的商务人士

渠道：公众号、小红书、详情页
```

AI 会分别输出三个版本，可直接参考演示视频：

## 三个场景串起来，就是一条完整的产品上新流水线

回到开头那个场景——新品要上架了，你需要一整套营销素材。

现在的工作流变成了：

```
随手拍一张产品照片
↓
场景一：生成启动页、横幅、海报 → 10 分钟搞定全套营销图
↓
场景二：精修商品图 + 生成展示视频 → 20 分钟搞定商品图和视频
↓
场景三：生成公众号/小红书/详情页文案 → 10 分钟搞定全渠道文案
```

**40 分钟，一个人，全套素材。** 以前需要设计师 + 摄影师 + 文案一起配合一周的工作量。

## 怎么开始？

### 环境准备

1. **安装 Qwen Code**：参考 [官方文档](https://qwenlm.github.io/qwen-code-docs/zh/users/overview/) 安装
2. **安装百炼 CLI**：参考 [百炼 CLI 文档](https://bailian.console.aliyun.com/cli) 安装
3. **启动 Qwen Code**：直接告诉 Qwen Code 你的 API Key，以及你的需求

### 安装技能包

打开 Qwen Code，直接说：

```
帮我安装 open1v-skills 技能包：git clone https://github.com/joeytoday/open1v-skills.git ~/.agents/skills/open1v-skills，然后安装依赖
```

安装完成后，说一句话就能触发完整流程：

- **生成商品图**：`帮我生成一张充电宝的白底主图`
- **写推广文案**：`帮我写一篇充电宝的推广文案，渠道：公众号、小红书、详情页`
- **发布到公众号**：`把公众号文案发布到草稿箱`

你也可以基于这些技能，定制更符合你们产品的版本。

### 接入飞书（可选）

想在飞书群里直接 @机器人完成以上操作？在 Qwen Code 里问一句：

```
帮我配置飞书频道，我想在飞书群里 @机器人 生成商品图和文案，并一步步告诉我步骤
```

Qwen Code 会引导你完成配置，5 分钟搞定。
